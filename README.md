## 목적
- [배틀라인](http://boardlife.co.kr/bbs_detail.php?bbs_num=581&tb=boardgame_strategy) 게임을 온라인으로 구현할 예정

## 사용 기술
- 웹 컴포넌트 : Vue.js
- CSS : flex box 를 적극 사용 할 예정
- Node.js

## 기능
- 우선 오프라인 구현(로테이션으로 할 수 있게 개발)
- 카드 드래그 드랍
- 특수 기능 카드는 우선 없음
- 간단한 셋 컬렉션 게임 형태로 구현
- 우선 이미지 없이 진행

## 우선 순위
- UI 레이아웃 구성
- 애니메이션
- 비즈니스 로직
